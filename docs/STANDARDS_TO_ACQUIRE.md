**Important:** Always acquire the latest version of standards. Check for:

## LICENSING NOTES

### Single User vs. Enterprise Licenses
- Single user licenses are typically for one person
- Enterprise licenses allow organization-wide access
- For AMI, recommend enterprise licenses for core standards

### Copyright Compliance
- Do not share purchased standards outside organization
- Each developer needing access should have licensed copy
- Free standards can be freely distributed
